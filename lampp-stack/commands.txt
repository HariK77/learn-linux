Update and upgrade, restart if needed

    $ sudo apt-get update
    $ sudo apt-get upgrade

    $ sudo apt-get install git
    $ sudo apt-get install curl

Install Apche2:

    $ sudo apt install -y apache2 apache2-utils
    $ sudo systemctl status apache2
    $ sudo systemctl start apache2
    $ sudo systemctl enable apache2
    $ apache2 -v

    Access http://localhost and check whether u are getting apache welcome page

If you want to enable firewall

    $ sudo ufw app list
    $ sudo ufw allow in "Apache Full"
    $ sudo ufw status

Giving permissions to user

    $ sudo chown $USER /var/www/html

Installing mariadb:

    $ sudo apt install mariadb-server mariadb-client

    $ sudo systemctl status mariadb
    $ sudo systemctl start mariadb
    $ sudo systemctl enable mariadb

    $ sudo mysql_secure_installation

    Press Enter

    Type Password and enter
    Re-type password Enter

    And press enter for all remaining questions

Creating a localhost use with all privileges:

    $ sudo mariadb -u root 
    It asks for system root password enter it
    Add a mysql user to authenticate to db from localhost usng below mysql quries

    CREATE USER 'maxx'@localhost IDENTIFIED BY 'maxx123';
    GRANT ALL PRIVILEGES ON *.* TO 'maxx'@localhost IDENTIFIED BY 'maxx123';
    FLUSH PRIVILEGES;

Enable Mod rewrite:

    $ sudo a2enmod rewrite
    $ sudo systemctl restart apache2

Enabling ssl (https)

    $ sudo a2enmod ssl
    $ sudo systemctl restart apache2
    $ sudo mkdir /etc/apache2/ssl
    $ sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout /etc/apache2/ssl/apache.key -out /etc/apache2/ssl/apache.crt

    the paths mentioned for .key and .crt file need to be used for sites which are going to use https

Enabling allow override All

    $ cd /etc/apache2/sites-available/
    $ sudo nano apache2.conf 

    change Allow override, None
    to Allow override, All

Add the virutal domains and hosts

    Adding domains

    $ sudo nano /etc/hosts

    Adding virual hosts

    $ cd /etc/apache2/sites-available/

    $ sudo nano default-ssl.conf 
    $ sudo nano 000-default.conf

    $ sudo a2ensite default-ssl.conf
    $ sudo a2ensite 000-default.conf

    $ sudo systemctl reload apache2

Installing zsh:

    $ sudo apt-get install zsh
    $ sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
    $ chsh -s /bin/zsh
    $ chsh -s $(which zsh)
    $ zsh --version
    $ echo $SHELL

    make zsh as default in terminal

    $ chsh -s /usr/bin/zsh

    zsh extensions:
    git clone https://github.com/zsh-users/zsh-autosuggestions.git $ZSH_CUSTOM/plugins/zsh-autosuggestions
    git clone https://github.com/zsh-users/zsh-syntax-highlighting.git $ZSH_CUSTOM/plugins/zsh-syntax-highlighting
    nano ~/.zshrc

    add to plugins: with space like below

    plugins=(git zsh-autosuggestions zsh-syntax-highlighting)
